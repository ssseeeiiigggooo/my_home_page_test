# MDXの記事ページ

ここは通常のマークダウンとして記述できます。
リストを作ったり、**太字**にしたりするのもいつも通りです。

{/* 💡 ポイント1: CSS Modulesをそのまま適用する */}
<div className={styles.box}>
  <p className={styles.title}>💡 補足情報</p>
  <p className={styles.content}>
    MDXの中では、HTMLタグを直接書くことができます！
    これによって、部分的にリッチなデザインを差し込むことが可能です。
  </p>
</div>
